# WidgetCraft

This is a Material You inspired app with a dynamic color theme that can be used to design great looking widgets for your phone. You can design widgets on your own in different shapes, with custom fonts for various purposes like weather, clock, trading, etc.

## What's This About

Basically, it's a widget builder. You can create custom widgets that look really nice with the Material You design system. Whether you want a weather widget, a clock, or something for trading, you can design it here and it'll match your phone's color scheme dynamically.

## Tech Stack

- **React Native** - for building the mobile app
- **TypeScript** - keeping things type-safe
- **Expo** - making development and deployment easier
